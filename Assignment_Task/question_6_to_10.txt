Q6. Write the code for adding new elements dynamically?

Q7. What is the difference between ViewState and SessionState? 
    The basic difference between these two is that the ViewState is to manage state at j
the client's end, making state management easy for end-user while SessionState manages 
state at the server's end, making it easy to manage content from this end too.


Q8. What is === operator? 
    This operator called equality comparison. It returns either true or false. The === 
operator compares operands and returns true if both operands are of the same data type 
and have same value, otherwise, it returns false.
